# 🌊 AQUAGYM (アクアジム)

**「運動を、育てる楽しみに。」** AQUAGYMは、あなたのトレーニング時間や日々の歩数を「デジタルの魚」の成長に変換する、フィットネス・ゲーミフィケーション・アプリです。

## ✨ コンセプト
単調になりがちなトレーニングを、キャラクター育成と結びつけることでモチベーションを維持します。「ジョギングフィッシュ」から「パワフルシャーク」まで、あなたの努力次第で水槽が賑やかになります。

## 🚀 主な機能
- **リアルタイム成長システム**: 運動を開始すると、卵から赤ちゃん、成魚へと段階的に成長します。
- **歩数ブースト**: 「今日の歩数」を入力すると、その日の成長スピードがアップ（最大x2倍以上）。
- **トレーニング図鑑**: 完走したメニューと育てた魚を記録し、いつでも振り返ることが可能。
- **カスタムメニュー**: 目標時間、カラー、成魚の見た目を自由に組み合わせて自分専用のワークアウトを作成。

## 🛠 使用技術
- **Frontend**: React (Hooks, Context)
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Build Tool**: Vite
- **Deployment**: Vercel
